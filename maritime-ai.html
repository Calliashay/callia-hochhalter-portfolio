Role

Data Analytics & Pipeline Architecture

Goal

Automate event detection in raw vessel tracking data.

Project Impetus

Current maritime tracking relies on heuristic models prone to manual error. This project explores the feasibility of integrating Generative AI to produce a more efficient, scalable, and accurate model for detecting significant maritime events.

The Technical Pipeline

The system transforms raw GPS data into actionable insights through a multi-stage process:

1. Data Ingestion & Resolution: Handles CSV, JSON, and NMEA radio messages, mapping varying column names to internal fields.
2. Event Detection Engine: Uses vectorized operations to detect speed-based events and course-based deviations.
3. AI Summarization: Integrated Claude AI to generate natural language summaries of complex movements for non-technical analysts.
4. Confidence Scoring: Quantitative reliability system achieving an average confidence score of 0.84.

Key Outcomes

  • Labeling Accuracy: 100% accuracy in classifying Route Deviations during testing.
  • Efficiency: Optimized memory usage by 40% during large-scale batch processing.
  • Actionable Insights: Transformed complex GPS data into high-fidelity dashboard.
