Executive Summary

SweepSlot is a text-based scheduling application designed to help small cleaning businesses organize daily operations. Built entirely in Python, it allows users to define services, set hourly availability, and book clients without the risk of double-bookings or time overlaps.

Technical Architecture

This project demonstrates how basic programming concepts can simplify business workflows without requiring advanced technical skills.

Conflict Resolution: Utilizes custom bookingOverlaps() and insideAnyAvailability() functions to ensure appointments fit within defined windows and don't conflict with existing jobs.
Session Persistence: Includes exportAllData and importAllData functionality, enabling users to save progress to a text file and reload it for future sessions.

Key Functions

Function Purpose
bookClient Validates times, checks availability, and computes final price based on duration.
toMinutes Converts "HH:MM" text into total minutes from midnight using manual loops and arithmetic.
askSweepAI Sends current session data and user questions to a local model via the buildSweepAIContext tool.

Lessons Learned

Developing SweepSlot highlighted the importance of organized data structures. By separating services, availability, and earnings into distinct lists, it became significantly easier to build robust helper functions. The addition of the GenAI "SweepSage" feature modernizes the tool while staying within the constraints of functional, beginner-level Python.
